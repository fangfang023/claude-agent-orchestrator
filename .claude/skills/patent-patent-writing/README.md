# Patent Writing Skill

专利撰写技能 - 帮助用户撰写符合中国专利申请标准的技术交底书。

## 功能说明

本技能提供结构化的工作流程，指导用户创建符合中国专利申请标准的技术交底书（技术公开书）。

## 使用场景

当您需要以下帮助时，可以使用此技能：
- 撰写技术交底书
- 申请发明专利
- 记录技术创新
- 准备专利申请文档

## 工作流程

本技能包含四个阶段：

1. **背景收集** - 了解发明、技术领域和创新点
2. **文档结构** - 按照专利标准构建必需章节
3. **内容撰写** - 撰写各章节的技术内容
4. **质量审查** - 验证完整性、一致性和符合性

## 输出文档

完整的技术交底书包含以下章节：

1. 发明名称
2. 技术领域
3. 背景技术
4. 发明内容
   - 要解决的技术问题
   - 技术方案
   - 有益效果
5. 附图说明（如有）
6. 具体实施方式
7. 权利要求书
8. 说明书摘要

## 使用方法

在Claude Code中，当您需要撰写专利时，可以：
- 直接说"帮我写专利"或"撰写技术交底书"
- 描述您的发明，系统会自动启动专利撰写工作流

## 注意事项

- 专利文档需要充分公开技术方案
- 术语必须全文保持一致
- 权利要求书是核心，限定保护范围
- 建议在专业专利代理人指导下完成最终申请
