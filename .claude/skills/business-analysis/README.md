# 商业价值评估与专利布局分析 Skill

## 概述

本 Skill 提供专业的技术交底书商业价值评估与专利布局分析，从企业战略、市场潜力、竞争格局等多维度分析技术的商业价值，并制定专利布局策略。

## 何时使用

**触发条件：**
- 用户要求进行"商业价值分析"、"商业分析"、"IP评估"
- 用户需要评估技术的市场潜力和商业价值
- 用户需要制定专利布局和申请策略
- 技术交底书生成后的商业评估

## 文件结构

```
business-analysis/
├── README.md          # 本文件
├── GENERATE.md        # 生成指导（详细报告模板）
└── REVIEW.md          # 评审标准
```

## 使用方式

### 生成商业分析报告

```
用户：基于这个技术交底书生成商业价值分析报告
→ 调用 business-analysis/GENERATE.md
→ 按照六个章节生成完整报告
```

### 审查商业分析报告

```
用户：检查这个商业分析报告的质量
→ 调用 business-analysis/REVIEW.md
→ 按照质量标准进行审查
```

## 输出规范

- **格式**：Markdown (.md)
- **命名**：`商业价值分析_[技术名称]_[YYYYMMDD].md`
- **编码**：UTF-8
- **输出位置**：`./generated_docs/`

## 特点

- ✅ 战略视角：从企业整体战略出发
- ✅ 全面分析：技术、市场、竞争、风险全覆盖
- ✅ 决策导向：为管理层提供明确的行动建议
- ✅ 专业深度：结合IP战略、市场分析、技术评估

## 角色设定

本 Skill 基于"企业知识产权总监（IP Director）"角色，具备：
- 10年以上企业IP管理经验
- 深厚的技术理解力
- 敏锐的市场洞察力
- 扎实的专利法知识
- 战略思维和商业敏感度

## 报告结构

生成报告包含六个主要章节：

1. **报告摘要** - 核心结论和关键行动建议
2. **技术核心与创新点剖析** - 技术先进性和壁垒评估
3. **商业价值评估** - 市场潜力、竞争优势、盈利模式
4. **专利布局策略建议** - 核心专利、外围布局、地域策略
5. **风险评估与规避** - 可专利性、FTO、技术泄露风险
6. **结论与行动方案** - 综合结论和具体行动清单

## 依赖关系

商业分析通常依赖以下文档：
```
技术交底书/技术方案 → 商业价值分析报告
                      → 决定是否申请专利
                      → 决定专利布局策略
```
